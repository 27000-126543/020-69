## 1. 产品概述

企业品牌公关团队专用的谣言扩散路径分析器，聚焦负面传言对品牌声誉的影响路径追踪与应对决策支持。产品通过监测清单自动汇总疑似谣言线索，以可视化路径复盘还原谣言传播链条，并提供结构化回应建议帮助团队在黄金时间内有序处置。

- 目标用户：企业品牌公关、危机管理团队成员
- 核心价值：将碎片化的谣言信息整合为可追溯的传播链路，从被动应对转向主动监测与有序处置

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用方式 | 核心权限 |
|------|----------|----------|
| 公关分析师 | 团队分配账号 | 录入监测清单、查看路径复盘、生成回应建议 |
| 公关主管 | 团队分配账号 | 全部分析师权限 + 审核回应建议、导出报告 |

### 2.2 功能模块

1. **监测清单**：品牌信息录入、排除词管理、每日谣言线索汇总与分类提示
2. **路径复盘**：谣言传播节点可视化、关键节点详情、热度趋势与受众画像
3. **回应建议**：平台账号联系清单、客服口径同步、证据材料匹配

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 监测清单 | 品牌信息录入 | 录入品牌名、产品名、门店城市、常见误称 |
| 监测清单 | 排除词管理 | 配置需要排除的普通投诉关键词，避免误报 |
| 监测清单 | 线索汇总看板 | 每日汇总疑似谣言线索，按类型分类展示：源头不明但转发快、老内容被重新包装、竞品话题夹带传播等 |
| 监测清单 | 线索详情卡片 | 每条线索展示摘要、来源平台、转发量、首次发现时间、风险等级标签 |

## 3. 核心流程

公关团队在使用本工具时，遵循"监测-复盘-应对"三步闭环流程：

1. 在监测清单中录入品牌相关信息和排除词
2. 系统每日自动汇总疑似谣言线索并分类提示
3. 公关人员发现高风险线索后，进入路径复盘查看传播链条
4. 在路径复盘中追溯关键节点，分析传播规律
5. 根据复盘结论进入回应建议，获取结构化应对方案
6. 按建议逐一执行并更新处置进度

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色系为主（深灰 #1a1a2e / 暗蓝 #16213e），搭配警示橙红（#e94560）作为强调色，传达专业危机管控氛围
- **辅助色**：冷灰（#0f3460）用于卡片背景，青绿（#53d8a8）用于正面指标，琥珀黄（#f0a500）用于中等风险
- **按钮风格**：圆角微弧（6px），扁平化设计，主要操作使用警示色填充，次要操作使用描边
- **字体**：标题使用 Noto Sans SC Bold，正文使用 Noto Sans SC Regular；数据数字使用 JetBrains Mono
- **布局风格**：左侧固定导航栏 + 右侧内容区，卡片式信息组织，信息密度较高但层次分明
- **图标风格**：线性图标（Phosphor Icons 风格），2px 线宽，与深色主题统一

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 监测清单 | 品牌信息录入 | 卡片式表单，深色输入框，品牌标签可视化展示 |
| 监测清单 | 排除词管理 | 标签云样式，可删除添加，灰色调区分 |
| 监测清单 | 线索汇总看板 | 网格卡片布局，风险等级色带（红/黄/蓝），数字用等宽字体突出 |
| 监测清单 | 线索详情卡片 | 展开式详情，来源图标、转发量动画计数、时间轴标记 |
| 路径复盘 | 传播路径可视化 | 横向时间轴+垂直节点图，节点用圆形标记，连线用渐变色表示传播强度，节点间有箭头指向 |
| 路径复盘 | 节点详情面板 | 右侧抽屉式面板，截图缩略图+放大查看，热度曲线用面积图，受众画像用环形图 |
| 路径复盘 | 热度趋势图 | 面积折线图，标注关键事件点，渐变填充 |
| 回应建议 | 平台账号清单 | 列表卡片，平台图标+账号名+优先级标签，一键复制联系方式 |
| 回应建议 | 客服口径同步 | 折叠面板，口径要点用要点列表，复制按钮 |
| 回应建议 | 证据材料匹配 | 图标+文字卡片，材料类型标签，推荐程度星级 |
| 回应建议 | 处置进度追踪 | 进度看板（未开始/进行中/已完成三列），可拖拽更新状态 |

### 4.3 响应式设计

- 桌面优先设计，最小支持 1280px 宽度
- 导航栏在窄屏下可折叠为图标模式
- 卡片网格在中等屏幕下从三列调整为两列
